# Life CFO

Life CFO is a household financial clarity app that helps families understand their money without overwhelm.

## What it does

Life CFO allows users to:

- Securely connect their bank accounts  
- View balances and recent transactions in one place  
- See a simple monthly cashflow picture  
- Track bills, recurring income, and planned spending  
- Make clearer financial decisions with less mental load  

## How we use Plaid

Life CFO uses Plaid to securely retrieve account and transaction data (with user consent).

Plaid data is used only to:

- Display account balances  
- Display transaction history  
- Power budgeting and cashflow summaries  

Life CFO:

- Does not sell user data  
- Does not use Plaid data for advertising  
- Does not initiate transfers or payments  
- Does not make lending or credit decisions
